DESCRIPTION

r.flip - Flip a raster map along the vertical or horizontal axis placed in the center of the computational region. The default flip is North-South which swaps positions along the horizontal axis. East-West flip can be done using -w flag which swaps along the vertical axis. A combined North-South and East-West flip can be done using -b flag.
